# CE-ESY-assignments

البرنامج  عبارة عن مصفوفة يتم التعامل معها بشكل دائري باستخدام مؤشرين:
واحد للكتابة (head) وآخر للقراءة (tail).

في البداية، يطلب البرنامج من المستخدم إدخال اسمه، ثم يقوم بإضافة النص "CE-ESY" إلى الاسم لتكوين سلسلة جديدة. بعد ذلك يبدأ التعامل مع هذه السلسلة حرفًا حرفًا.

يتم إنشاء مخزن دائري بحجم معين، ثم تُكتب الأحرف داخله باستخدام دالة مخصصة. كل مرة يتم إدخال حرف، يتحرك مؤشر الكتابة خطوة للأمام، وإذا وصل إلى نهاية المصفوفة يرجع إلى البداية (بسبب العملية الدائرية).

إذا امتلأ المخزن، تتوقف عملية الكتابة ولا يتم إدخال باقي الأحرف.

بعد الانتهاء من الكتابة، يبدأ البرنامج بقراءة الأحرف من المخزن بنفس الفكرة، حيث يتحرك مؤشر القراءة تدريجيًا حتى يتم إخراج جميع القيم.

تم تطبيق هذه العملية مرتين:

مرة باستخدام مخزن صغير لا يكفي لتخزين كل النص
ومرة باستخدام مخزن أكبر يستطيع استيعاب النص كاملًا
